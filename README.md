# 开源阅读器

一个跨平台的开源电子书阅读软件，支持多书源、在线阅读与本地缓存下载。

适配多种阅读数据源，支持自定义源。

## 功能特性

- 基于 wxPython 开发，界面轻量与易扩展
- 适配 Reader 源  
 参考项目: https://github.com/binbyu/Reader  
- 支持自定义书源格式
- 支持 Windows 与 Linux

---

## 自定义书源

支持配置与导入自编写的书源
